Nom de code : Chu Chu Rocket

On est sur une grille de 19*10 cases. Chaque case peut être un mur ou une case vide. Il y a des robots sur cette carte. Le but du jeu est de garder en vie les robots le plus longtemps possible.

Pour cela, le joueur doit placer des flèches sur la carte. Il peut poser autant de flèche qu'il le souhaite.

Les robots ne font qu'avancer dans une direction. Quand elles arrivent sur une flèche, elles changent de direction en fonction de la flèche. Au premier tour les robots vont dans une direction qui est donnèe dans les inputs.

Un robot meurt si :

  • Il rentre dans un mur
  • Il rentre dans un autre robot
  • Il commence une boucle infinie

Pour calculer le score de joueur, chaque tour rapporte autant de point qu'il y a de robots sur la carte. Au début on a donc 10 points par tour, puis 9 points quand un robot aura réussi à s'échapper, etc ... Le problème est donc ici le chemin le plus long. Un problème connu en informatique mais surtout connu pour ne pas avoir de solution.

La difficulté du jeu : éviter la boucle infinie
Si un robot entre dans une boucle infinie, il est retirée du jeu

Le code ne joue qu'une seule fois. Il reçoit tout les inputs, il doit donner toutes les flèches à poser en une seule ligne. La partie se déroule toute seule jusqu'à la fin.

INPUTS

  • 1 ligne : Un entier pour le nombre de ligne à lire
  • X lignes suivantes : la carte du jeu. Un . indique une case vide. Un # indique un mur. Un U si c'est une flàche vers le haut. Un R si c'est une flàche vers la droite. Un D si c'est une flàche vers le bas. Un L si c'est une flàche vers la gauche.
  • 1 ligne : Le nombre de robots
  • X lignes suivantes : une ligne contenant les 3 informations suivantes dans l'ordre x, y, direction.
    • x : la position x du robot
    • y : la position y du robot
    • direction : la direction du robot. U pour UP, R pour RIGHT, D pour DOWN et L pour LEFT
OUTPUTS

Une seule ligne qui contient autant de triplet qu'on veut séparé par des espaces. Chaque triplet doit être de la forme x y type. Chaque triplet permet de poser une flèche sur une case. Les types possibles sont donc U, R, D et L

Exemple d'output : 8 9 U 13 10 D 15 1 L
